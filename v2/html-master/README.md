# HTML Master

This folder will contain the approved HTML prototype for each page.

## Rule

The approved HTML is the master design source for Elementor.

## Planned pages

- `home/`
- `products/`
- `industries/`
- `services/`
- `resources/`
- `about/`
- `contact/`

## Section approval process

For every page section:

1. Review the existing reference page.
2. Confirm changes.
3. Implement the section in HTML.
4. Preview and test the section.
5. Record QA notes.
6. Approve and lock the section.

## Current reference

The existing `pages/home/index.html` file is the starting reference for homepage content, layout and design direction.
