# QA Checklist

Use this checklist before a page or section is marked approved.

## Visual QA

- Desktop layout checked.
- Tablet layout checked.
- Mobile layout checked.
- Section spacing checked.
- Card alignment checked.
- Button styling checked.
- Image cropping checked.
- SVG scale and alignment checked.
- Header and navigation checked.
- Footer links checked.

## Content QA

- Headings are clear and page-specific.
- Body copy matches TShirtSwiss positioning.
- CTAs are consistent.
- No placeholder text remains.
- No duplicated sections remain.
- Links point to the correct destination.

## Technical QA

- HTML validates reasonably.
- CSS has no obvious conflicting rules.
- Images include useful alt text where appropriate.
- Decorative icons are treated as decorative.
- Pages load without console errors.
- Relative paths work from the page location.

## Accessibility QA

- Keyboard navigation works.
- Focus states are visible.
- Colour contrast is acceptable.
- Navigation labels are meaningful.
- Heading order is logical.
- Forms have labels.
