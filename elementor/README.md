# Elementor Folder Structure for TShirtSwiss.ch

This folder contains the planned Elementor template and component architecture for the TShirtSwiss.ch website.

## Purpose
- `templates/`: holds Elementor section and template documentation for page sections such as hero, trust, process, FAQ and CTA.
- `components/`: holds smaller Elementor component blueprints for cards, stats, buttons, and reusable blocks.

## Recommended usage
- Use these folders to document Elementor container layouts, global section templates, and reusable widget patterns.
- Match these folders with Elementor Theme Builder templates when the site is built.

## Current folders
- `templates`
- `components`
