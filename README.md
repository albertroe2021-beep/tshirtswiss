# WordPress Folder Structure for TShirtSwiss.ch

This folder contains the planned WordPress page architecture for the TShirtSwiss.ch website.

## Purpose
- `pages/`: holds page-level folders that correspond to site pages and page types.
- `templates/`: holds reusable template groups for global layouts, sections and components.

## Recommended usage
- Use `pages/` to organize the WordPress site structure and map content for WPML translations.
- Use `templates/` to store reusable Elementor template documentation, section blueprints and global layouts.
